# Cookie Consent Banner

We love cookies. This website uses cookies to improve your experience. 

A lightweight, vanilla JavaScript cookie consent solution with persistent storage.

## Features

- **Persistent Storage**: User preferences saved using localStorage API
- **ES6 Modules**: Uses import/export syntax for modular code architecture
- **Notification System**: Displays confirmation messages about user preference
- **No Dependencies**: Pure vanilla JavaScript, no frameworks required

## Technologies
- HTML5
- CSS3
- JavaScript (Modern ES6)
- LocalStorage API
